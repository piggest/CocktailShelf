# CocktailShelf

カクテルレシピ検索の Web サービス。426種のカクテルを日本語で検索・閲覧できる。

公開可能とマークされた画像のみ表示する。未マークのカクテルはアイコンプレースホルダーで表示される。

## 機能
- 名前 / 材料での AND 検索（複合フィルタ対応）
- スタイル（ショート / ロング / ロック / ショット / フローズン / ホット / コーヒー / スパークリング / ソフトドリンク）と ベース別の絞り込み
- 材料一覧タブ（出現頻度順）
- お気に入り（押下回数カウンタ）
- 飲んでみたいリスト

## ローカル起動

```sh
python3 -m http.server 8000
```

`http://localhost:8000` を開く。
