メスカル・マルガリータ

Mezcal Margarita

ショートアルコールロックグラスTequila ベース

材料

メスカル45 ml
トリプルセック22 ml
ライムジュース22 ml
アガベシロップ5 ml
塩リム用
カットライムガーニッシュ

作り方

ロックグラスの縁をライムで湿らせ塩をつけてスノースタイルにする。シェーカーにメスカル、トリプルセック、ライムジュース、アガベシロップを入れ氷とともによくシェイクし、氷入りのグラスへ注ぎライムウェッジを添える。マルガリータのテキーラをメスカルに替えた、燻製香が際立つ大人向けの一杯。
