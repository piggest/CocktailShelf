ラモス・ジン・フィズ

Ramos Gin Fizz

ロングアルコールハイボールグラスGin ベース

材料

ジン45 ml
レモンジュース30 ml
シュガーシロップ30 ml
生クリーム60 ml
卵白1
バニラエッセンス2 滴
炭酸水20 ml

作り方

シェイカーにジン、レモンジュース、シュガーシロップ、生クリーム、卵白、バニラエッセンスを入れて氷なしで強くドライシェイクし、氷を加えてさらにしっかりシェイクする。氷を入れたグラスに注ぎ、冷えた炭酸水で満たす。長時間振ることで生まれるクリーミーな泡が特徴の、ニューオーリンズクラシック・フィズ。
