フレーミング・ボルケーノ

Flaming Volcano

ショートアルコールRum ベース

材料

ライトラム15 ml
ブランデー15 ml
オーバープルーフラム8 ml
オレンジジュース60 ml
レモンジュース30 ml
オルジェ・シロップ30 ml

作り方

オーバープルーフラム以外をクラッシュドアイスとともにブレンダーで撹拌し、ボルケーノボウルへ注ぐ。中央のくぼみにラムを注いで点火し、炎を眺めながらストローで吸う。トロピカル風味の共有用ドリンクで、ティキバーの名物。
