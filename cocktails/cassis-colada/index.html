カシス・コラーダ

Cassis Colada

ロングアルコール11oz ゴブレット　340　(331ml)Liqueur ベース

材料

ココナッツリキュール45 ml
クレーム・ド・カシス10 ml
パイナップルジュース45 ml
牛乳45 ml

作り方

シェーカーにココナッツリキュール、クレーム・ド・カシス、パイナップルジュース、牛乳と氷を入れてよくシェイクし、グラスに注ぐ。角氷を加える。ピニャ・コラーダにカシスを加えた、トロピカルでクリーミーな一杯。
